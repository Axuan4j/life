# Life 用户 H5

- 技术：Vue 3 + Vant 4 + Pinia + Vue Router + Axios
- 当前能力：
  - 点选验证码登录
  - 首页 Feed
  - 发现页与搜索结果页
  - 消息中心
  - 发帖、评论、点赞、转发
  - 帖子详情
  - 我的主页与他人主页
- 启动：
  - `pnpm install`
  - `pnpm dev`
- 构建：
  - `pnpm build`
- 说明：
  - 本地默认通过 Vite 的 `/api` 代理转发到“本机局域网 IP:18080”，优先避开本机 SSH/隧道对 `127.0.0.1:18080` 的占用
  - 如果本地用户端后端不在 `18080`，可先设置 `VITE_DEV_API_TARGET`
  - 生产默认走站点同源 `/api`
  - 只有特殊场景才需要显式注入 `VITE_API_BASE_URL`
